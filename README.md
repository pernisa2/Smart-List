## 📝 Sobre o Projeto

O **Lista Smart** nasceu da necessidade de otimizar a experiência de idas ao supermercado e o controle de pendências rotineiras. Diferente de anotações comuns em papel, o aplicativo categoriza itens dinamicamente, monitora o volume de pendências em tempo real e garante que o usuário nunca perca seus dados, mesmo se fechar o navegador por engano.

## 🚀 Diferenciais Técnicos e Funcionalidades

Este projeto demonstra maturidade no desenvolvimento Front-End através de recursos essenciais:

* **Persistência de Dados Automatizada (`LocalStorage`):** Utiliza a API de armazenamento local do navegador para salvar, ler e sincronizar o estado da lista instantaneamente a cada adição ou alteração, eliminando a necessidade de um botão manual de "salvar".
* **Separação por Categorias:** Permite organizar os itens (ex: Mercado, Farmácia) de forma modular, facilitando a navegação visual no momento das compras.
* **Filtros de Estado Dinâmicos:** Sistema de abas interativas para alternar rapidamente entre a visualização de "Todas", "Pendentes" ou tarefas "Feitas".
* **Contador de Pendências em Tempo Real:** Feedback imediato ao usuário indicando a quantidade exata de itens que ainda precisam ser atendidos.
* **Suporte a PWA (Progressive Web App):** Configurado para ser instalável diretamente em dispositivos mobile (como iPhone e Android), operando com fluidez e visual nativo de aplicativo.
